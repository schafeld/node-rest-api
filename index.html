apiAPI Configuration

This demo application can connect to a REST API server. For local development, make sure your API server is running on http://localhost:3000.

Supported operations:
• GET - View all items
• POST - Create new items
• PUT - Update existing items
• DELETE - Remove items
• Reset - Restore original data
